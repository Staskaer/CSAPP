# 第二章：信息的处理与表示

### __编程要求/规则__：

> > _假设_
> > 
> > 1. 整数用补码形式表示
> > 2. 有符号数的右移是算数右移
> > 3. 数据类型`int`是 _w_ 位的。对于某些题目，会给定 _w_ 的值；但是在其他情况下，只要 _w_ 是8的整数倍，代码就可以运行。可以用表达式 `sizeof(int)<<3`来计算 _w_ 。
> 
> > _禁止使用_
> > 
> > + 条件语句、循环、分支条件、函数和宏调用
> > + 除法、模运算和乘法
> > + 比较运算符
> 
> > _允许的运算_
> > 
> > + 所有的位级和逻辑运算
> > + 左移和右移，但是位移量只能在 `0 - w `位之间
> > + 加法和减法
> > + 相等，不等测试（有些题目也不允许）
> > + 整形常量 ‘ INT_MIN ’ 和 `INT_MAX`
> > + 对 `int` 和 `unsigned` 执行强制类型转换
